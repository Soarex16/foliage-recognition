Инструкция по запуску веб-приложения:
1. Установить docker
2. Установить docker-compose
3. Открыть консоль
4. Перейти в текущую директорию
5. Выполнить команду docker-compose up

Приложение будет доступно по адресу http://localhost:3000/
Для тестирования можно использовать изображения из папки images_multi

Jupyter блокноты в которых велись эксперименты расположены в директории notebooks.
